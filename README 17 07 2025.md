# MySalesApp

A modern React Native application for Playmobil sales reps.  
It provides an efficient workflow for field order taking, syncing with Firestore, and exporting orders for office processing.

---

## ✨ Features Implemented

### 🔹 Modern Order Flow

- **Order Flow in 5 Steps:**
  1. **Customer Selection:**  
     - Search by customer name, VAT, or code.
     - Manual entry for new/unlisted customers.
     - Stores selected customer in order context.
  2. **Product Selection:**  
     - Fast scrollable list of all products.
     - Quantity selection with Material-style +/- box.
     - Quantity can be entered via keyboard or buttons.
     - Product info visible per row.
     - Context-safe state so changes persist if navigating back.
     - Card and UI polish for mobile, Material look, safe areas respected.
  3. **Order Review:**  
     - Shows all selected products.
     - Change/delete quantities.
     - Collapsible product list.
     - Select payment method (with discount calculation).
     - Notes field, Material input, always visible above keyboard.
     - FAB (floating action button) for confirmation.
     - Totals (net, discount, VAT, final) calculated and shown.
  4. **Order Summary:**  
     - Customer info.
     - Product lines and order totals.
     - Notes and payment method.
     - Export to XLSX (file share).
     - (Planned: Export to PDF)
     - Firestore sync (planned, stubs exist).
  5. **Export/Confirmation:**  
     - Success/error message.
     - Back to Home, View Order.

### 🔹 State Management & Data

- **OrderContext:**  
  Manages the full order state, persists selected customer, product lines, payment, and summary through all steps.
- **Safe area:**  
  All screens displayed within safe area to avoid notification/softkey overlay.
- **Keyboard aware:**  
  Keyboard input fields always visible.
- **Material Design:**  
  Color palette, button styles, cards, and list layouts.
- **UI Polish:**  
  Collapsible product lists, sticky titles, FABs, proper spacing.

    - Orders:  
      - Save/export orders to Firestore on final export.
      - Show sync status, offline/online handling.
      - Allow resync on failure.
    - Customers/Products:  
      - Sync full customer/product lists from Firestore with local DB.
      - Manual "sync now" trigger from UI.

2. **Order Management & Editing**
    - View/Filter/Edit past orders (local & remote).
    - Edit and resume "draft" (unsent) orders.
    - Allow deleting or copying previous orders.

3. **PDF Export**
    - Implement PDF export (using `react-native-html-to-pdf`).
    - Match summary screen format for PDF layout.

4. **GPS/Location Capture**
    - Capture location when starting order.

5. **Role-based Access**
    - Allow users to see only their own orders unless "manager" access.

6. **UI/UX Polish**

    - Add tests for key business logic (order math, data sync, export).

## 🚀 Getting Started

1. `npm install`
2. Run Metro:  
   `npx react-native start --reset-cache`
3. In another terminal:  
   `npx react-native run-android`
4. (Or run on iOS if you have a Mac)
